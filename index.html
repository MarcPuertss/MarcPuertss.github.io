Hex AI

An AI-based Hex Project developed in Java to play the strategic board game Hex. The AI employs sophisticated algorithms to make optimal real-time decisions. It uses the Minimax Algorithm enhanced with Alpha-Beta Pruning for efficient move evaluation, alongside Iterative Deepening Search (IDS) to ensure timely responses. The Dijkstra Algorithm serves as a heuristic to calculate the shortest paths on the board, helping evaluate winning strategies. With this combination of technologies, IA HEX delivers a challenging opponent and showcases the practical application of artificial intelligence in game theory and decision-making.

Technologies: Java, Minimax Algorithm, Alpha-beta Pruning, Iterative Deepening Search (IDS), Djikstra Algorithm

Public events, similar to an Instagram feed, could include activities like meet-ups or walks created by pet owners, and other users could join them. Connect Four Game Module

In this project, I created a Connect Four game module in Java, utilizing a framework provided by my professor. This project focused on implementing a strategic AI capable of making optimal decisions during gameplay. The core of the AI relies on the Minimax Algorithm with Alpha-Beta Pruning, which ensures efficient decision-making by pruning unnecessary branches in the search tree, significantly improving performance. The implementation demonstrates the practical application of algorithmic problem-solving in game AI, offering a challenging and dynamic opponent.

Technologies: Java, Minimax Algorithm, Alpha-Beta Pruning
