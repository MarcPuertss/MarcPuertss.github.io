Works

Decentralized Voting System

I envisioned developing a decentralized voting system using blockchain technology, but for this project, I implemented a conventional database to explore what kind of data would need to be stored. This helped me understand the structure required for a future blockchain implementation.

Objectives
  • Create an intuitive interface to manage large sets of data.
  • Develop a script to generate and populate a database with millions of records.
  • Simulate a secure voting system where votes can be reviewed on a "blockchain".
  • Create two user roles: one who can generate votes (administrator) and another who can vote (member).
What I Learned

One of the most valuable lessons I learned in this project was how to efficiently handle large amounts of data. By developing a script that generated millions of records and implementing a system capable of managing these volumes, I gained a deep understanding of the challenges and strategies involved in large-scale data management.

Technologies Used
  • Node.js
  • Express.js
  • Vue.js
  • PostgreSQL

  • Python
